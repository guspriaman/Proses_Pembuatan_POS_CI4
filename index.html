Pada Kesempatan ini saya akan menjelaskan proses pembuatan apk POS (Poin Of Sales) yang telah saya buat beberapa waktu yang lalu, Pembuatan aplikasi Ini Mengunakan bahasa Pemmograman javascrips, php, html, codeignite 4 dan tamplate admin LTE 3.

yang pertama adalah kita lakukan adalah,
1. mengistral Visual Studio Code sebagai code editor itu menulis program pembuatan apl
2. mengistral Xampp sebagai perangkat lunak berbasis web server yang bersifat open source (bebas)
3. mengistral Composer di dalam Visual studio Code
4. mengistral Codeignite 4 di dalam Visual studio Code dengan Perintah > composer create-project codeigniter4/appstarter --no-dev


5. mendownload Template admin LTE 3
6. file admin LTE 3 di masukan kedalam folder project aplikasi POS di dalam folder public dengan nama file assets
7. untuk memulai penulisan program,
8. folder Pos di buka di dalam Visual Studio Code
9. untuk menjalan program aplikasi POS di browser atau crome. kita harus menuliskan perintah php spark serve di dalam terminal VCS perintah ini akan selalu di lakukan setiap memulai penulisan program

10. menjalankan program di browser dengan perintah localhost:8080


11. Merubah nama file env menjadi .env
12. Merubah perintah CI_ENVIRONMENT = production menjadi CI_ENVIRONMENT = development yang berada di dalam folder .env
13. Merubah beberapa perintah pada folder .env seperti pada gambar di samping

folder .env sebelum di ubah

folder .env setelah di ubah

14. Membuat file htaccess di dalam folder aplikasi POS
